# 🚀 Portfolio no Ar - Vercel Deploy

## Deploy 1-click:
1. `npm install` (instala Vercel CLI)
2. `vercel login` (conta Vercel)
3. `vercel` (deploy automático!)

**URL gerada**: https://potifi-portfolio.vercel.app

## Estrutura:
```
.
├── portfolio.html
├── css/style.css
├── js/script.js
├── package.json
└── vercel.json
```
